# 俄罗斯方块 (Tetris)

## 项目描述

这是一个使用 C++ 语言和 EGE 图形库实现的经典俄罗斯方块游戏。  
项目学习和巩固面向对象、文件操作、运算符重载和异常处理等C++核心概念。

## 核心功能

-   **经典玩法**：不同形状的方块随机下落，玩家可以移动和旋转方块。
-   **消行计分**：填满的行会自动消除并得分。
-   **等级系统**：随着得分增加，游戏等级提升，方块下落速度加快。
-   **方块预览**：提前告知玩家下一个将出现的方块。
-   **高分纪录**：游戏会自动记录并加载历史最高分。
-   **读取难度**：可以从 "difficulty.txt" 文件配置初始游戏难度。
-   **游戏结束提示**：明确的游戏结束界面。
-   **运算符重载应用**：
    -   `++` / `--` 控制方块左右移动。
    -   `+=` 用于游戏分数更新。
-   **异常处理**：
    -   对潜在的游戏逻辑错误进行检测和处理。
    -   对文件读取等操作提供一定的容错性。

## 环境

-   **编程语言**：C++
-   **操作系统**：Windows 11
-   **图形库**：EGE (Easy Graphics Engine)
-   **开发环境**：Visual Studio 2022

## 文件结构

-   `main.cpp`: 程序入口，初始化游戏对象并运行游戏循环。
-   `Tetris.h` / `Tetris.cpp`: 包含游戏主逻辑类 `Tetris` 的定义和实现。
-   `Blocks.h` / `Blocks.cpp`: 包含方块基类 `Blocks` 的定义和实现，以及派生出的各种具体形状方块类。
-   `record.txt`: (运行时自动生成/使用) 存储历史最高分。
-   `difficulty.txt`: (需手动创建/配置) 存储游戏初始难度级别 (如 0, 1, 2)。

## 操作说明

-   `左箭头 (LEFT)`: 向左移动方块
-   `右箭头 (RIGHT)`: 向右移动方块
-   `上箭头 (UP)` : 旋转方块
-   `下箭头 (DOWN)`: 加速方块下落 (软降)
